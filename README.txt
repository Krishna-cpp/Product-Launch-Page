# Project: Product Launch Page

## Overview
This project is a one-page, front-end product launch website for a fictional gadget, built using only HTML and CSS. The goal was to create a visually appealing and responsive landing page that highlights product features, testimonials, and a clear call-to-action.

## Features
- A hero section with a product image and headline.
- A section for key product features with icons.
- A testimonials section with customer feedback.
- A pricing table with two different plans.
- A prominent call-to-action button.

## Technologies Used
- HTML5 for semantic structure
- CSS3 for styling and responsive layout (using Flexbox and Grid)
